# Revisoo App

FastAPI backend + Next.js frontend for [tryrevisoo.vercel.app](https://tryrevisoo.vercel.app) - the platform that hosts public study note pages generated by the [Revisoo script](https://github.com/Revisoo/revisoo-script).

---

## Stack

| Layer | Tech |
|---|---|
| Frontend | Next.js 14 (App Router), deployed on Vercel |
| Backend | FastAPI + SQLAlchemy, deployed on Railway |
| Database | PostgreSQL (Neon) |

---

## Local development

### Backend

```bash
cd backend
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env   # fill in DATABASE_URL (Neon connection string)
alembic upgrade head
uvicorn app.main:app --reload
```

Backend runs at `http://localhost:8000`. API docs at `http://localhost:8000/docs`.

### Frontend

```bash
cd frontend
npm install
```

Create `frontend/.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:8000
```

```bash
npm run dev
```

Frontend runs at `http://localhost:3000`.

## API reference

| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/api/v1/users/claim` | Claim a username |
| `GET` | `/api/v1/users/check/{username}` | Check username availability |
| `POST` | `/api/v1/notes` | Push a note (used by the script) |
| `GET` | `/api/v1/notes/{username}` | List notes for a user |
| `POST` | `/api/v1/streaks/record-view` | Record a page view and update streak |
| `GET` | `/api/v1/leaderboard` | Get ranked leaderboard |

## Deployment

**Frontend (Vercel)**
- Connect the repo, set root directory to `frontend`
- Add environment variable: `NEXT_PUBLIC_API_URL=https://your-railway-url.up.railway.app`
- Do **not** mark this variable as Sensitive - it must be available at build time

**Backend (Railway)**
- Connect the repo, set root directory to `backend`
- Add environment variable: `DATABASE_URL=your_neon_connection_string`

## Project structure

```
revisoo-app/
   backend/
      app/
         api/v1/routes/   # users endpoints
         routers/         # notes, streaks, leaderboard endpoints
         models/          # SQLAlchemy ORM models
         services/        # business logic
         schemas.py       # Pydantic request/response models
         main.py          # FastAPI app entry point
      migrations/         # Alembic migrations
   frontend/
      src/app/
         page.tsx          # landing page
         claim/            # /claim
         leaderboard/      # /leaderboard
         notes/[username]/ # public notes feed
      src/lib/api.ts        # typed API client
```
